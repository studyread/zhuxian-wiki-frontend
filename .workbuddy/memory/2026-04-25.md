## AI知识库搜索优化

### 问题
用户反馈：词条中有"云梦宗新手入门攻略"，但AI无法回答。

### 原因
1. 旧代码只搜索 title/summary/tags 字段，不搜索 content 字段
2. MySQL LIKE 搜索不支持中文分词，"云梦宗"整体匹配失败

### 解决方案
1. **修改 KnowledgeEntryMapper.xml**
   - 增加 content 字段搜索
   - 新增 searchByKeywords 分词搜索方法
   - 优化排序：标题匹配优先

2. **修改 AIService.java**
   - 新增 extractKeywords() 中文分词逻辑
   - 使用多关键词 OR 匹配搜索
   - fallback 机制：分词搜索结果少时补充单关键词搜索

3. **数据库直接插入**
   - 通过 SQL 插入正确的 UTF-8 中文词条（ID=10）
   - 解决 API 调用时的编码问题

### 关键代码
```java
// extractKeywords 示例
"云梦宗新手入门攻略" → ["云梦宗新手入门攻略", "云梦宗新手", "云梦宗", "云梦", "新手入门", ...]
```
