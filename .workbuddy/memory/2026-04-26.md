# 2026-04-26 工作日志

## 诛仙世界Wiki项目更新

### 新增功能

1. **用户独立点赞权限**
   - 新增 `user_article_like` 表（user_id, article_id 联合唯一索引）
   - 后端：UserArticleLike 实体、Mapper、XML
   - ArticleService 新增 `likeArticleWithUser()` 和 `hasUserLikedArticle()` 方法
   - ArticleController 新增 `POST /api/articles/{id}/like/user` 和 `GET /api/articles/{id}/like/status`
   - 前端点赞改用 `articleApi.likeWithUser()` 和 `articleApi.getLikeStatus()`

2. **AI助手用户独立会话**
   - AIController 新增 `/api/ai/history` 接口（支持 userId 参数）
   - AIService 新增 `getChatHistoryByUserId()` 方法
   - 前端 AIChat.vue 登录用户使用 `X-User-Id` header，游客使用普通 sessionId

3. **AI输出样式优化**
   - formatMessage() 支持：标题(h2/h3/h4)、引用块、代码块、列表、加粗斜体
   - 消息气泡样式优化：圆角、阴影、渐变背景
   - 用户消息：朱砂红渐变背景 + 白色文字
   - AI消息：深色代码块、引用块高亮

### 修改的文件
- 前端：`src/api/article.js`, `src/views/Article.vue`, `src/views/Home.vue`, `src/views/admin/AdminLayout.vue`, `src/views/AIChat.vue`
