# 诛仙世界 Wiki 部署说明

## 一键部署

### 前置要求
1. Windows 10+（自带 OpenSSH）
2. Maven、Node.js 已安装并加入环境变量
3. 服务器已配置 SSH 密钥登录（免密码）

### 使用方法

**方式一：双击运行（带默认提交信息）**
```powershell
# 在文件资源管理器中，右键 deploy.ps1 -> 使用 PowerShell 运行
```

**方式二：命令行运行（自定义提交信息）**
```powershell
cd F:\诛仙世界wiki
.\deploy.ps1 -CommitMsg "feat: 新增XX功能"
```

### 脚本执行流程
1. **Git 提交** → 自动检测前后端改动，推送到 GitHub
2. **后端打包** → `mvn clean package -DskipTests`
3. **前端构建** → `npm run build`
4. **上传服务器** → scp 上传 JAR 和静态文件
5. **重启服务** → ssh 远程执行重启命令

### 首次配置 SSH 免密登录
```powershell
# 生成密钥（如果还没有）
ssh-keygen -t rsa -b 4096

# 复制公钥到服务器
ssh-copy-id root@47.98.240.202

# 测试免密登录
ssh root@47.98.240.202
```

### 手动部署（备用）

如果脚本有问题，可以分步手动执行：

```powershell
# 1. Git 提交
cd F:\诛仙世界wiki\zhuxian-wiki-backend
git add . && git commit -m "update" && git push

cd F:\诛仙世界wiki\zhuxian-wiki-frontend
git add . && git commit -m "update" && git push

# 2. 后端打包
cd F:\诛仙世界wiki\zhuxian-wiki-backend
mvn clean package -DskipTests

# 3. 前端构建
cd F:\诛仙世界wiki\zhuxian-wiki-frontend
npm run build

# 4. 上传（替换为你的用户名）
scp target/zhuxian-wiki-backend-1.0.0.jar root@47.98.240.202:/opt/zhuxian-wiki/
scp -r dist/* root@47.98.240.202:/opt/zhuxian-wiki/html/

# 5. 服务器重启
ssh root@47.98.240.202 "cd /opt/zhuxian-wiki && pkill -f zhuxian-wiki && nohup java -jar zhuxian-wiki-backend-1.0.0.jar > app.log 2>&1 &"
```

### 常见问题

**Q: PowerShell 执行策略阻止运行脚本？**
```powershell
# 以管理员身份运行 PowerShell，执行：
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

**Q: scp/ssh 命令找不到？**
- Windows 10+ 自带 OpenSSH，如果找不到：
- 设置 → 应用 → 可选功能 → 添加功能 → OpenSSH 客户端

**Q: 上传时提示输入密码？**
- 需要配置 SSH 密钥免密登录，见上文"首次配置"

**Q: 服务器重启后服务没起来？**
```bash
# SSH 登录服务器查看日志
ssh root@47.98.240.202
cd /opt/zhuxian-wiki
tail -f app.log
```
